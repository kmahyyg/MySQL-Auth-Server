# MySQL-Auth-Server

The purpose of this tool is to Act like "MySQL Server", And return a row with the query values.

## Getting Started

```
git clone http://github.com/diefunction/MySQL-Auth-Server.git
```

### Examples
```
service mysql stop
pkill -9 mysql
```
then
```
python3 MySQL-Auth-Server.py
```
